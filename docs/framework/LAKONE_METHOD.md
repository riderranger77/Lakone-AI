# Lakone Method v1.0

> A practical framework for solving problems with AI.

## Philosophy

AI is not a magic tool. AI is a thinking partner.

Output quality depends on:

- Clear objective
- Good context
- Correct model
- Structured prompt
- Verification
- Iteration

## The Lakone Method

### 1. Define

Ask: What problem am I trying to solve?

Output: one clear objective.

### 2. Collect

Gather:

- Context
- Data
- Constraints
- Examples
- Desired output

### 3. Select

Choose the best AI model or tool for the task.

Examples:

- Coding: Claude, ChatGPT
- Research: ChatGPT with web/search tools
- Writing: Claude, ChatGPT
- Image: Dreamina, Seedream
- Video: Hailuo, Kling

### 4. Prompt

Create a structured prompt:

1. Role
2. Context
3. Goal
4. Requirements
5. Output format
6. Examples

### 5. Generate

Produce the first draft. Never assume the first output is final.

### 6. Review

Check:

- Accuracy
- Logic
- Completeness
- Bias
- Hallucination
- Formatting

### 7. Iterate

Improve repeatedly until the result meets the goal.

### 8. Save

Document:

- Prompt
- Result
- Lesson learned
- Reusable template

## Golden Rules

1. Never skip context.
2. Never trust AI blindly.
3. Always verify important facts.
4. Reuse successful prompts.
5. Improve continuously.
