# Tokens

Tokens are the basic units of text processed by AI language models.

## What Is a Token?

A token can be a word, part of a word, number, symbol, punctuation mark, or space pattern.

For example, a sentence may be split into several tokens before it is processed by an AI model.

## Why Tokens Matter

Tokens affect:

- Input length
- Output length
- Context window usage
- API cost
- Processing speed
- Model performance

## Token Budget

A token budget is the total number of tokens available for the input, instructions, conversation history, and response.

## Best Practices

- Keep prompts clear and efficient.
- Remove unnecessary text.
- Put the most important context first.
- Summarize long documents before deeper analysis.
- Use structured prompts to reduce waste.

## Common Mistakes

- Giving too much irrelevant context
- Asking for very long outputs without structure
- Forgetting that conversation history also uses tokens
- Not summarizing repeated information
