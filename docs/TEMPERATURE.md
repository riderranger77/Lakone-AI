# Temperature

Temperature is a setting that controls how creative or deterministic an AI model's output may be.

## Low Temperature

Low temperature usually produces more focused, consistent, and predictable answers.

Best for:

- Facts
- Summaries
- Code
- Structured output
- Business writing

## High Temperature

High temperature usually produces more creative, varied, and exploratory answers.

Best for:

- Brainstorming
- Creative writing
- Idea generation
- Alternative phrasing

## Practical Guidance

- Use low temperature for accuracy and consistency.
- Use medium temperature for balanced writing.
- Use higher temperature for creative exploration.

## Common Mistakes

- Using high temperature for tasks that need precision
- Expecting temperature to fix unclear prompts
- Ignoring output verification
