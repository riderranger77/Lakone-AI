# Context Window

The context window is the maximum amount of information an AI model can consider at one time.

## What It Includes

The context window can include:

- System instructions
- User prompts
- Conversation history
- Uploaded text or documents
- Tool results
- The model's generated response

## Why It Matters

A larger context window allows the model to work with more information, but it does not automatically guarantee better answers.

## Best Practices

- Put the most important information near the prompt.
- Summarize long background material.
- Remove irrelevant details.
- Use headings and sections.
- Ask the model to focus on specific parts of the context.

## Common Mistakes

- Assuming the model remembers everything forever
- Providing too much unorganized information
- Mixing unrelated tasks in one prompt
- Not prioritizing key facts
