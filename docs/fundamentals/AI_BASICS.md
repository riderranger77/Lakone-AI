# AI Basics

## Apa itu AI?

Artificial Intelligence adalah teknologi yang membuat komputer mampu menjalankan tugas yang biasanya membutuhkan kecerdasan manusia.

Contoh:

- Menjawab pertanyaan
- Menulis teks
- Menganalisis data
- Membuat gambar
- Membuat video
- Membantu coding
- Merangkum dokumen

## Apa itu LLM?

LLM atau Large Language Model adalah model AI yang dilatih dengan banyak teks sehingga mampu memahami dan menghasilkan bahasa manusia.

Contoh LLM:

- ChatGPT
- Claude
- Gemini
- DeepSeek
- Qwen
- Llama
- Mistral

## Token

Token adalah potongan teks yang dibaca oleh model AI. Satu kata bisa terdiri dari satu atau beberapa token.

## Context Window

Context window adalah batas jumlah informasi yang bisa diingat model dalam satu percakapan atau satu permintaan.

## Temperature

Temperature mengatur tingkat kreativitas jawaban.

- Rendah: lebih stabil dan konsisten
- Tinggi: lebih kreatif tetapi bisa kurang akurat

## Embedding

Embedding adalah cara mengubah teks menjadi angka agar bisa dibandingkan secara semantik oleh komputer.

## Vector Database

Vector database menyimpan embedding agar dokumen bisa dicari berdasarkan makna, bukan hanya kata kunci.

## RAG

Retrieval-Augmented Generation adalah teknik menggabungkan pencarian dokumen dengan kemampuan generasi AI.

## Function Calling

Function calling memungkinkan AI memanggil fungsi atau tool tertentu untuk menjalankan tugas.

## AI Agent

AI agent adalah sistem AI yang dapat merencanakan langkah, menggunakan tool, dan menyelesaikan tugas secara lebih mandiri.
