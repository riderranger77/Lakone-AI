# Prompt Engineering

## What is Prompt Engineering?

Prompt engineering is the skill of writing clear instructions for AI so it can produce better, more accurate, and more useful results.

---

## Basic Prompt Formula

A good prompt usually contains:

1. Role
2. Task
3. Context
4. Output format
5. Constraints

---

## Example

```text
Act as an AI writing assistant.
Help me rewrite this paragraph to sound professional.
Keep the meaning the same.
Use simple English.
