# Prompt Engineering

Prompt engineering adalah seni dan teknik menyusun instruksi agar AI menghasilkan jawaban yang lebih tepat.

## Struktur Prompt yang Baik

Prompt yang baik biasanya memiliki:

1. Role
2. Context
3. Goal
4. Constraints
5. Output format
6. Examples

## Template Dasar

```text
You are [role].

Context:
[context]

Goal:
[goal]

Requirements:
- [requirement 1]
- [requirement 2]

Output format:
[format]
```

## Prinsip Penting

- Berikan konteks yang cukup.
- Jelaskan tujuan dengan jelas.
- Tentukan format output.
- Beri batasan jika perlu.
- Minta AI memeriksa hasilnya.
- Iterasi hasil pertama.

## Kesalahan Umum

- Prompt terlalu pendek
- Tidak memberi konteks
- Tidak menentukan output
- Meminta terlalu banyak hal sekaligus
- Tidak memverifikasi hasil
