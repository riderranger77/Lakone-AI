# Contributing to Lakone-AI

Thank you for your interest in contributing to Lakone-AI.

## How to Contribute

You can contribute by improving documentation, adding prompts, creating workflows, fixing typos, or sharing practical AI examples.

## Content Guidelines

- Use clear and beginner-friendly language.
- Keep examples practical and reusable.
- Use Markdown format.
- Place files in the correct folder.
- Avoid copyrighted or private content.

## Folder Guidelines

- `docs/` for AI learning materials.
- `prompts/` for prompt templates.
- `workflows/` for step-by-step AI workflows.
- `examples/` for practical use cases.
- `templates/` for reusable formats.
- `tools/` for helper scripts or tool notes.
- `assets/` for images and supporting files.

## Pull Request Checklist

Before submitting a pull request, make sure:

- The file name is clear.
- The content is useful.
- The formatting is readable.
- The contribution matches the purpose of this repository.

## Community Standard

Be respectful, helpful, and focused on practical AI learning.
