# Universal Prompts

Universal prompts are designed to work across multiple AI platforms such as ChatGPT, Claude, Gemini, Grok, and Copilot.

## Purpose

Use this folder for general prompt templates that are not tied to one specific AI model.

## Contents

- Master system prompt
- Master user prompt
- Reusable prompt template
- General best practices

## Goal

Create prompts that are clear, flexible, reusable, and easy to adapt for different AI tools.
