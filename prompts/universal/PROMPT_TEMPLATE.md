# Universal Prompt Template

Use this structure to create reusable prompts.

## Template

```text
Task:
[Describe what you want the AI to do]

Context:
[Provide background information]

Goal:
[Define the desired result]

Constraints:
[List rules, limitations, tone, length, language, or format]

Output Format:
[Describe how the answer should be structured]

Examples:
[Optional examples]
```

## Checklist

Before using a prompt, check that it has:

- Clear task
- Enough context
- Specific goal
- Important constraints
- Expected output format

## Tip

A good prompt is not always long. A good prompt is clear, specific, and useful.
