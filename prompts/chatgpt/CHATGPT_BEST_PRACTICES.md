# ChatGPT Best Practices

## 1. Give Clear Context

Explain the background, goal, and audience before asking for output.

## 2. Define the Role

Example:

```text
Act as a senior marketing strategist.
```

## 3. Specify the Output Format

Example:

```text
Give the answer as a table with columns: problem, solution, priority.
```

## 4. Use Iteration

Ask ChatGPT to improve, simplify, expand, or compare the answer.

## 5. Add Constraints

Useful constraints include:

- Language
- Tone
- Length
- Format
- Target audience
- Tools used

## 6. Ask for Examples

Examples help make the answer easier to understand and apply.

## Simple Prompt Formula

```text
Act as [role].
Help me with [task].
Context: [context].
Constraints: [constraints].
Output format: [format].
```
